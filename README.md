# Advit Pandey Portfolio

Minimal high-motion AI/ML portfolio for GenAI, agentic AI, and applied machine learning roles.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Three Fiber / Three.js
- Lucide icons

## Local Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
```

Vercel should auto-detect Vite:

- Build command: `npm run build`
- Output directory: `dist`

## Notes
